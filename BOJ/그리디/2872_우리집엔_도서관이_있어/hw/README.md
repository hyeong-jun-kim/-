## ❗ 풀이 방법
1. main에서 배열의 길이, 배열을 입력받고 solution에 넘겼다.
2. num에 최댓값을 넣고 뒤에서부터 탐색하면서 최댓값이라면 count를 1 증가시켜주고, 최댓값을 하나씩 감소시켜주었다.(제대로 정렬되어있는 원소의 갯수)
3. 전체 원소의 갯수 - 제대로 정렬되어있는 원소의 갯수를 하면 정답이 나온다.
