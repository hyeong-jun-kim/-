## ❗ 풀이 방법

1. 톱니바퀴 상태를 저장하는 배열과 방향을 담고있는 배열을 생성한다. 
2. 톱니바퀴를 돌리는 rotate함수를 만들어서 각 방향에 따라 rotate 시킨다. 
3. check 함수를 생성해서 문제에서 주어진 톱니바퀴 기준으로 좌측 우측 톱니바퀴의 방향성을 검사한다.
